		JavaScript
		----------
	JavaScript is a object-based scripting programming language.

	it is a interpreted language

	it is used to create interactive websites.

applications of javascript:
--------------------------
	client-side validations
	display the date and time
	pop-up,alert messages
	...
	...

we can implement the javascript code in 3-ways,they are

	1).script tag in our html tag of body of the webpage
		(inline javascript)/between the body tag
                              

	2).script tag in our head of the webpage
		(internal javascript)/between the headtag

	3).to create seperate javascript file
			(external javascript)


ex: inline javascript
---
	sample.html
	-----------
<!DOCTYPE html>
<html>
<head>
</head>
<body>
<script>
  alert('hello siva')
</script>
<h1>welcome to Vagdevi Technologies</h1>
</body>
</html>

ex: internal javascript
---
	sample.html
	-----------
<!DOCTYPE html>
<html>
<head>
<script>
    function msg(){
	alert("hello world");
       }
</script>
</head>
<body>
<button onclick=msg()>ClickMe</button>
<h1>welcome to Vagdevi Technologies</h1>
</body>
</html>

ex: external javascript
---
	sample.html
	----------
<!DOCTYPE html>
<html>
<head>
<script type="text/javascript" src="demo.js">    
</script>
</head>
<body bgcolor="powderblue">
<button onclick=msg()>ClickMe</button>
<h1>welcome to Vagdevi Technologies</h1>
</body>
</html>

	demo.js
	--------
function msg(){
	alert("hello world");
       }
